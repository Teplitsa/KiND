KiND – Accessibility-Focused Theme for Media and Non-Profits

== Description ==

Introducing KiND, a versatile and open-source WordPress theme designed specifically
for independent media outlets and non-profits, events, and special projects.
This customizable theme is built with a focus on accessibility, adhering to WCAG standards.
With its user-friendly interface and compatibility with popular plugins,
KiND empowers you to create appealing, functional, and secure websites that perfectly suit your needs.
Join a community-driven development approach and unlock the potential of your media website
or non-profit with KiND —a flexible solution for your online presence.
